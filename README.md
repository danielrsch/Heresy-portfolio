Welcome to the official repository for my personal portfolio. This project serves as a digital archive of my works, skills, and technical journey.

## Overview

This portfolio is designed with an emphasis on **clarity**, **simplicity**, and **impactful code**. It utilizes a clean, reading-focused aesthetic inspired by editorial design, paired with subtle nods to the *Astartes* archives for that extra bit of character.

### The why?
- **Minimalist **: Focus on typography and whitespace for maximum readability.
- **Responsive Design**: Optimized for all viewports, from desktop to mobile.
- **Smooth Interactions**: Sophable CSS transitions and macOS-inspired UI elements.
- **Thematic Flair**: A unique blend of professional structure and geeky personality.
